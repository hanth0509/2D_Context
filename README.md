# HTML5 Canvas Game Development

## Introduction

This project explores the fundamentals of game development using HTML5 Canvas and JavaScript. Throughout the project, core concepts such as rendering, animation, physics, collision detection, and sprite-based graphics are implemented to create interactive browser games.

## 1. Set-up an HTML5 Canvas

The first step is learning how to create and configure an HTML5 Canvas. This includes understanding the canvas element, obtaining a 2D rendering context, and working with the coordinate system used for drawing graphics.

## 2. Draw Shapes, Paths and Texts

Canvas provides a variety of drawing tools for creating game elements. Basic shapes, custom paths, and text rendering are introduced to build visual objects directly on the canvas.

## 3. Game Loop

A game loop is the foundation of every real-time application. Using `requestAnimationFrame()`, the project continuously updates game objects and redraws the scene to create smooth gameplay.

## 4. Canvas Animation

Animation is achieved by updating object properties over time. By incorporating Delta Time calculations, movement remains consistent regardless of frame rate, resulting in smoother and more reliable animations.

## 5. Collision Detection and Physics

Basic physics concepts are implemented to simulate realistic object interactions. This includes gravity, velocity, mass, restitution, and collision detection between objects within the game world.

## 6. Images and Sprite Animations

To improve visual quality, images and sprite sheets are used instead of simple geometric shapes. Sprite animation techniques allow objects to display different frames, creating more dynamic and engaging game elements.
